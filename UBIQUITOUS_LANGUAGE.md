# ユビキタス言語 (Ubiquitous Language)

タスクタイマーアプリにおける用語の定義です。

| 用語 (日本語)        | 用語 (英語)     | 定義                                                                                                                       |
| :------------------- | :-------------- | :------------------------------------------------------------------------------------------------------------------------- |
| **やることリスト**   | **Todo List**   | 一度に実行する一連の「やること」のリスト。                                                                                 |
| **やること**         | **Todo**        | やらないといけないタスク。あらかじめ決められた固定の時間を持ちます。                                                       |
| **ごほうび**         | **Reward**      | やることリストの最後に１つだけ設定可能なタスク。「やること」が予定より早く終わると、その分このタスクの実行時間が増えます。 |
| **どーなつタイマー** | **Donut Timer** | 「やること」や「ごほうび」の残り時間を視覚的に表す、ドーナツ型の円形ゲージ。                                               |
