# ユビキタス言語 (Ubiquitous Language)

どーなつタイマーで使う主要ユーザー用語の定義です。  
正規用語は子ども向け日本語を採用し、英語・型名は別名として扱います。

## 基本概念

| 正規用語（日本語）   | 別名・実装語（英語/型）                 | 定義                                                                               | 関連箇所                         |
| :------------------- | :-------------------------------------- | :--------------------------------------------------------------------------------- | :------------------------------- |
| **やることリスト**   | Todo List, `TodoList`, `DomainTodoList` | 一度に実行する「やること」と「ごほうび」をまとめた単位。                           | `src/types.ts`, `src/domain/...` |
| **やること**         | Todo, `Task(kind='todo')`               | 子どもが順番に進める通常タスク。予定時間を持つ。                                   | `src/types.ts`, `src/domain/...` |
| **ごほうび**         | Reward, `Task(kind='reward')`           | リスト内で最後に配置する特別タスク。やることが早く終わると、実行可能時間が増える。 | `src/types.ts`, `src/domain/...` |
| **どーなつタイマー** | Donut Timer, `TimerSettings`            | 残り時間を形と色つきのゲージで表現する視覚タイマー。                               | `src/components/timer/*`         |
| **シンプル表示**     | Simple List View                        | やることリストをアイコンと分表示で確認する、リスト選択画面の表示方式。             | `src/components/task/*`          |

## 実行モード

| 正規用語（日本語）   | 別名・実装語（英語/型）         | 定義                                                                 | 関連箇所                         |
| :------------------- | :------------------------------ | :------------------------------------------------------------------- | :------------------------------- |
| **ごほうびモード**   | Reward Mode, `mode='duration'`  | ごほうび時間を「やることの早く終わった分」で増やす基本モード。       | `src/domain/timer/policies/*`    |
| **目標時刻モード**   | Target Time Mode, `target-time` | 「何時何分に終える」を基準に、今使えるごほうび時間を逆算するモード。 | `src/types.ts`, `src/domain/...` |
| **ひとりモード**     | Single Mode, `timerMode=single` | 1つのリストを1画面で進めるモード。                                   | `src/storage.ts`, `src/App.tsx`  |
| **きょうだいモード** | Sibling Mode, `sibling-0/1`     | 2つのリストを左右に分けて同時進行するモード。                        | `src/storage.ts`, `src/App.tsx`  |

## 時間概念

| 正規用語（日本語） | 別名・実装語（英語/型）        | 定義                                                     | 関連箇所                           |
| :----------------- | :----------------------------- | :------------------------------------------------------- | :--------------------------------- |
| **予定時間**       | Planned Time, `plannedSeconds` | タスクにあらかじめ設定した時間。                         | `src/types.ts`, `src/domain/...`   |
| **経過時間**       | Elapsed Time, `elapsedSeconds` | 実行中にカウントされる現在までの時間。                   | `src/types.ts`, `src/domain/...`   |
| **実績時間**       | Actual Time, `actualSeconds`   | タスク完了時点で確定する、実際にかかった時間。           | `src/types.ts`, `src/domain/...`   |
| **残り時間**       | Remaining Time                 | 予定時間（またはごほうび時間）から経過時間を引いた時間。 | `src/components/task/TaskCard.tsx` |

## 状態・進行

| 正規用語（日本語）   | 別名・実装語（英語/型）        | 定義                                                   | 関連箇所                            |
| :------------------- | :----------------------------- | :----------------------------------------------------- | :---------------------------------- |
| **実行中**           | Running, `status='running'`    | タイマーが進行している状態。                           | `src/types.ts`, `src/domain/...`    |
| **一時停止**         | Paused, `status='paused'`      | 途中で停止し、再開可能な状態。                         | `src/types.ts`, `src/domain/...`    |
| **やすむ（ボタン）** | Pause Button, `label='やすむ'` | 実行中のタイマーをいったん止める子ども向け操作ラベル。 | `src/components/timer/Controls.tsx` |
| **完了**             | Done, `status='done'`          | タスクが終わり、結果（実績時間）が確定した状態。       | `src/types.ts`, `src/domain/...`    |
