# どーなつタイマー 🍩

どーなつタイマーは、お子様の主体性を大切にしながら、目標に対して前向きに取り組めるよう支えるビジュアルタイマーです。
お子様には「楽しい」「うれしい」「やってみたい」を、保護者の方には時間の見通しと前向きな声かけのきっかけを届けることを目指しています。
親子でいっしょにやることを考えながらも、お子様自身が「やってみよう」と思える流れを重視し、4歳前後で順番管理や時間管理が苦手なお子様の日常支援にも配慮しています。

## 🎯 コンセプト

自己決定理論の考え方をもとに、お子様の主体性を第一に考えています。
保護者の方が一方的に押し付けるのではなく、お子様が自ら目標を選んだり決めたりする補助になることを大切にし、始めやすい、続けやすい、終わりやすい流れの中で小さな「できた」を重ねられる体験を支えます。

## ✨ 特徴

- **楽しいビジュアル**: 時間が経つにつれてドーナツが少しずつ欠けていき、お子様には「進んでいる」が楽しく見えます。
- **ごほうびモード**: 早く終わった分だけ「ごほうび（あそぶ時間）」が増える仕組みで、「終わったらうれしいことがある」を親子で共有しやすくします。余ったドーナツがごほうびへ飛んで合流するアニメーションで、増えたことがひと目で分かります。
- **目標時刻モード**: 「◯時◯分に終わらせる」という目標から逆算して、今遊べる時間を自動計算します。時計を読む力より、納得して進める流れを大切にしています。
- **きょうだいモード**: 二人同時に、画面分割で別々のタスク・タイマーを進められます。家族全体の進み方をそろえやすく、声かけも前向きに保ちやすくなります。
- **シンプルなリスト表示**: やることリストはアイコンと「分」表示で見やすく確認できます。
- **カスタマイズ**: タイマーの形（星型や六角形など）や色を自由に変更でき、お子様に合う「楽しい」「やってみたい」を見つけやすくなります。
- **保護者向けガイド**: ご家庭で目標を決めて前向きに取り組むための考え方、主な機能、安心して使える点をアプリ内ですぐ確認できます。
- **プライバシー安心**: データはブラウザ内（LocalStorage）にのみ保存され、サーバーには送信されません。

## 🚀 使い方

1. 「やることリスト」を作成または選択します。
2. 順番にタスクを開始します。
3. すべての「やること」が終わると、余った時間でたっぷり遊べます！
