# どーなつタイマー 🍩

お子様が楽しく「やること」を終わらせられるようにデザインされた、ドーナツ型のビジュアルタイマーです。

## ✨ 特徴

- **楽しいビジュアル**: 時間が経つにつれてドーナツが少しずつ欠けていく視覚的なタイマー。
- **ごほうびモード**: 早く終わった分だけ「ごほうび（あそぶ時間）」が増える仕組みで、やる気を引き出します。
- **目標時刻モード**: 「◯時◯分に終わらせる」という目標から逆算して、今遊べる時間を自動計算します。
- **きょうだいモード**: 二人同時に、画面分割で別々のタスク・タイマーを進められます。
- **カスタマイズ**: タイマーの形（星型や六角形など）や色を自由に変更可能。
- **プライバシー安心**: データはブラウザ内（LocalStorage）にのみ保存され、サーバーには送信されません。

## 🚀 使い方

1. 「やることリスト」を作成または選択します。
2. 順番にタスクを開始します。
3. すべての「やること」が終わると、余った時間でたっぷり遊べます！

## 🛠 技術スタック

- React 19
- TypeScript
- Vite
- Framer Motion (アニメーション)
- Lucide React (アイコン)

## 📦 開発

```bash
# インストール
make install

# 開発サーバー起動
make dev

# ビルド
make build

# フォーマット
make format

# Lint
make lint

# テスト
make test

# プレビュー
make preview
```
